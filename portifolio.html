projeto calendario

Esse projeto se trata de um tipo de agenda web, onde o cliente pode marcar algum horário disponível e o adm pode gerenciar esse horário. Páginas de login e cadastro, essa aplicação web tem uma página de cadastro e duas de login, a de cadastro e onde o usuário pode criar sua conta, a primeira página de login e onde o usuário entra com sua conta e é redirecionado para a página principal, já a segunda página de login é a do adm, onde ele coloca seu nome e sua senha, que é colocada por mim mesmo diretamente no banco de dados, é é redirecionado para a página de administração de horários.
Já as páginas após o login o cliente é redirecionado para a página principal, onde ele vai escolher o dia e clicar nele, após clicar no dia irá aparecer os horários disponíveis para aquele dia, os que estiverem disponíveis terão um botão que o usuário clica e agende um horário, caso o horário já estiver ocupado o botão não estará disponível, quando o usuário estiver procurando um dia para marcar o horário e esse dia já tiver passado, ou for sábado e domingo o dia não estará disponível, no caso só aparece uma mensagem falando da indisponibilidade.
já a página principal do adm é um pouco diferente, ela consiste em uma estrutura bem parecida, porém as coisas mudam quando você vai na parte dos horários disponíveis, lá terá uma duas caixas, uma para colocar o nome do cliente que deseja o horário e seu número de celular, já que as fezes é necessário mudar o horário, entre outras coisas, além de ter o botão de deletar, que limpa o horário que o adm desmarcar e o deixa livre novamente.
